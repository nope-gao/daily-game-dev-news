# GameDev & AI Daily Express

> 游戏开发 · 游戏圈 · AI圈发展 — 每日精选速递

每日自动搜索三大领域的最新动态，生成 HTML 日报推送至 GitHub Pages 和微信。

**在线阅读**：https://jason-gao.github.io/gamedev-ai-daily/

## 内容板块

| 板块 | 覆盖范围 |
|------|----------|
| 🎮 游戏开发 | Godot/Unity/Unreal 引擎更新、开发工具、技术教程、开源项目 |
| 🕹️ 游戏圈 | 新游发布、行业事件、Steam 数据、独立游戏推荐 |
| 🤖 AI圈发展 | 大模型发布、AI 工具、框架更新、行业突破 |

## 技术栈

- **内容生成**：Hermes Agent + AI 模型
- **页面**：纯静态 HTML + CSS，报纸风格
- **图表**：Mermaid.js
- **部署**：GitHub Pages
- **自动化**：Hermes Cron Job + 微信推送
